# 🚦 Smart Traffic Sign Detection System

## 📌 Overview

This project is an AI-powered system that detects and classifies traffic signs using deep learning.
It helps in building intelligent driver assistance systems for safer and smarter transportation.

---

## 🎯 Features

* 🔍 Real-time traffic sign detection
* 🧠 Deep learning-based classification (CNN / YOLO)
* 📊 High accuracy on traffic sign dataset
* 🎥 Works on images and video input
* ⚡ Fast and efficient performance

---

## 🛠️ Tech Stack

* Python
* OpenCV
* PyTorch
* NumPy
* Matplotlib
